## Typography Hierarchy

```
H1 (Hero):        8rem (128px)   Space Grotesk Bold
H2 (Sections):    4.5rem (72px)  Space Grotesk Bold
H3 (Cards):       1.5rem (24px)  Space Grotesk Bold
Body Large:       1.25rem (20px) Inter Light
Body:             1rem (16px)    Inter Regular
Body Small:       0.875rem (14px) Inter Light
```

## Animation Patterns

### On Scroll Reveal
```
Initial:  opacity: 0, y: 20
Animate:  opacity: 1, y: 0
Duration: 0.6s
```

### Stagger Children
```
Container delay: 0
Each child:      +0.1s delay
```

### Hover Effects
```
Scale:    1.02
Y-offset: -5px
Duration: 0.3s
```

### Background Animations
```
Blobs:    Scale pulse (8s infinite)
Marquee:  Translate X (20s linear)
Dots:     Float up/down (6s infinite)
```

---

## Responsive Breakpoints

### Mobile (< 640px)
- Single column layout
- Smaller typography
- Stacked navigation (hamburger menu recommended)
- Reduced padding

### Tablet (640px - 1024px)
- 2 column grid
- Medium typography
- Horizontal navigation
- Moderate padding

### Desktop (1024px+)
- 3 column grid
- Full typography scale
- All features visible
- Maximum padding

---

## Micro-Interactions

1. **Header**
   - Blur increases on scroll
   - Active section highlights
   - Smooth scroll to anchors

2. **Buttons**
   - Arrow shifts right on hover
   - Scale slightly on press
   - Background darkens

3. **Cards**
   - Lift up on hover
   - Border brightens
   - Gradient overlays fade in

4. **Images**
   - Zoom 110% on hover
   - External link icon appears
   - Overlay darkens

5. **Icons**
   - Scale 110% on hover
   - Rotate slightly
   - Shadow intensifies

---

## Visual Mood

**Dark & Confident**
- Deep blacks create drama
- High contrast makes content pop
- Neon accents feel futuristic

**Smooth & Fluid**
- Every transition eased
- Animations never jarring
- Flow between sections seamless

**Modern & Professional**
- Geometric typography
- Ample white space
- Asymmetrical but balanced

**Energy & Motion**
- Constant subtle movement
- Pulsing elements
- Floating particles
- Infinite scrolls

---

This design creates a **confident, avant-garde tech brand** that feels both professional and cutting-edge.
